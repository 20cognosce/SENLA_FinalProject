Для установки и запуска программы нужно:

1) Открыть проект внутри IntellijIdea Ultimate
1.1) java -version >= 11 (у меня стоит openjdk version "11.0.2" 2019-01-15)
1.2) Проверить переменные окружение JAVA_HOME, CATALINA_HOME, M2_HOME
1.3) Установить папку для логов в config.properties
2) Создать новую базу данных в pgAdmin (имя БД - final_project, user - postgres, password - password123)
2.1) postgres -V >= 14.4
3) Установить соединение c БД внутри IntellijIdea
4) Запустить скрипт ddl-script.sql
5) Запустить скрипт dml-script.sql
