ЗАПУСК ИСХОДНИКОВ В INTELLIJ IDEA:

1) Открыть проект внутри IntelliJ IDEA Ultimate
В терминале убедиться, что
1.1) java -version >= 11 (у меня стоит openjdk version "11.0.2" 2019-01-15)
1.2) mvn -version >= 3.8 (у меня 3.8.6)
1.3) catalina version >= Apache Tomcat/8.5.81
1.4) При проблемах проверить системные переменные окружения JAVA_HOME, CATALINA_HOME, M2_HOME, Path.
1.5) При необходимости установить свою папку для логов в logback.xml,
путь по умолчанию - ${Домашняя-директория-пользователя}/logs/logging.log (например C:\Users\cognosce\logs\logging.log)

2) Создать новую базу данных в pgAdmin (имя БД - final_project, user - postgres, password - password123)
2.1) postgres -V >= 10

3) Установить соединение c БД внутри IntelliJ IDEA
3.1) Запустить скрипт ddl-script.sql
3.2) Запустить скрипт dml-script.sql

4) В конфигурации запуска внутри Intellij IDEA настроить Local Tomcat Server, не забыв выбрать war-архив проекта
как артефакт для развертывания.
